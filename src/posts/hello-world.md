this is the first post on my new site.

i set this up with [eleventy](https://www.11ty.dev/) so i can write
posts in markdown and they get rendered automatically. no databases,
no javascript frameworks, just markdown files and a build step.

## why this stack

i wanted something minimal:

- write posts as `.md` files
- build to static html
- deploy anywhere (netlify, github pages, a vps, whatever)
- keep the same terminal aesthetic
